# ABSA Cloud Project

## Deskripsi
Proyek ini merupakan aplikasi web berbasis cloud untuk analisis sentimen berbasis aspek (Aspect-Based Sentiment Analysis) pada review film IMDb.

## Arsitektur Singkat
Sistem menggunakan AWS dengan komponen:
- EC2 untuk backend
- RDS untuk database
- S3 untuk penyimpanan model
- Load Balancer untuk distribusi traffic
- CloudFront untuk CDN

## Teknologi
- Python (Flask/FastAPI)
- Machine Learning (Naive Bayes, SVM, Random Forest)
- AWS Cloud
